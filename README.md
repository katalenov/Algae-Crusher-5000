# Algae-Crusher-5000
This repository contains all Arduino firmware and Python-based GUI code used to operate the Algae Crusher 5000.

The Algae Crusher 5000 is a mechanical briquetting press designed to densify microalgal blooms. Its purpose is to help address the growing challenge of harmful algal blooms (HABs) in Maine and around the world by converting algal biomass into a manageable, transportable form.

This project is developed in accordance with the senior capstone design sequence in Mechanical Engineering at the University of Maine. Updates will be posted regularly as development continues. If you encounter bugs or issues, please report them so our team can address them as quickly as possible.
